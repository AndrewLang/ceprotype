Keysight Command Expert is a faster and easier way to program instruments.

  • Seamless integration with Excel, VEE, SystemVue, MATLAB, Visual Studio, Python, and LabVIEW
  • Easy to find, use, and view full documentation for SCPI, IVI-C and IVI-COM commands
  • Simplifies the process of retrieving measurement data from instruments

Quick Start

Examples

Initializes channel 1 and enables the output of an Agilent N6700B power supply. 

 

Instructions for Running:

1. Enter your instrument address in the Connect step.

2. Run the sequence.

 

Operation:

1. Reset the power supply.

2. Set voltage to 3 volts and current to 1.5 amperes. 

3. Set voltage and current protection. 

4. Turn the output of channel 1 on.

5. Wait for output to settle.

6. Measure the voltage.

7. Reset the power supply, turning off the output.

 

Requirements:

This example was written for the N6700B power supply.  This example was tested with the following instrument(s): N6700B(ver C.00.02).

Operates the N6705A DC Power Analyzer to step to 10 Volts on channel 1 when triggered, and record 10 voltage samples after being triggered.

 

Instructions for Running:

1. Enter your instrument address in the Connect step.

2. Run the sequence.

 

Operation:

1. Set the instrument up to step it's channel 1 output voltage when triggered.

2. Set the channel 1 output voltage to be 10 volts when triggered.

3. Set the output voltage on channel 1 to 5 volts. 

4. Turn the output of channel 1 on.

5. Set the instrument output voltage to be triggered by a remote interface trigger command.

6. Set the instrument to measure 10 points of data for the output voltage on channel 1 when triggered.

7. Set the period between measurement points to be 10 microseconds.

8. Set the instrument measurement system to be triggered by a remote interface trigger command.

9. Initiate the measurement triggering system.

10 Initiate the the output triggering system.

11. Trigger the instrument.

12. Fetch the measured data (output voltage on channel 1) as an array of floating point numbers.

 

Restrictions and Requirements:

This example was written for the N6705A Power Analyzer.  This example was tested with the following instrument(s): N6705A (ver C.01.05).
